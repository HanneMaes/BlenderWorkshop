ğŸ‘ï¸ Blender downloaden

ğŸ‘ï¸ Navigatie

Bij het opstarten krijg je een splash screen te zien. Klik ergens op het scherm om dit weg te doen.

Binnen Blender kan je navigeren op 2 manieren:

  1. Via het Navigation Gizmo:

2. Via de muis en het toetsenbord

  • Roteren: midden muisknop
  • Zoomen: scroll
  • Pannen (bewegen): shift-midden muisknop

ğŸ‘ï¸ Primitieven

Primitieven verwijderen

Blender opent altijd met een Cube. Deze verwijder je door X in te drukken, of door Object > Delete in het menu te selecteren.

Primitieven plaatsen

Nieuwe objecten plaats je via de shortcut Shift-A, of via het Add menu. Voor deze opdracht hebben we enkel objecten van het type Mesh nodig.

Plaats nu zelf een 2 objecten, een met de shortcut Shift-A en een andere via Add > Mesh.

Positie, schaal en rotatie

Verplaatsen, herschalen en roteren die je via deze shortcuts:

  • Verplaatsen: G (Grab of grijpen)
  • Herschalen: S (Scale)
  • Roteren: R (Rotate)

Positie, schaal en rotatie over 1 as

Wil je deze bewerking uitvoeren over 1 as, dan druk je de toets in van de as.

  • X-as: X
  • Y-as: Y
  • Z-as: Z

ğŸ‘ï¸ Smooth & flat shading

Wanneer je een object plaats ziet dit er heel scherp uit, maar je kan het er ook rond uit laten zien. Dit noemt men Shading. Er zijn 2 soorten Shading, Flat & Smooth.

Je kan de Shading van een object aan passen door in het menu Object > Shade Smooth of Object > Shade Flat te selecteren.
